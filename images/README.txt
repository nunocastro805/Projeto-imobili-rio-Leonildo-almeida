INSTRUÇÕES PARA USAR SUAS PRÓPRIAS IMAGENS, VÍDEOS E LOCALIZAÇÕES

Para usar suas próprias imagens, vídeos e localizações:

1. IMAGENS E VÍDEOS:
   - Coloque suas imagens na pasta public/images/ com estes nomes:
     - apartamento1.jpg, apartamento2.jpg, apartamento3.jpg (para o primeiro imóvel)
     - casa1.jpg, casa2.jpg (para o segundo imóvel)
     - video1.mp4 (vídeo do primeiro imóvel)

2. LOCALIZAÇÕES:
   - Ao criar um novo imóvel no Dashboard, preencha o campo "Localização"
   - Exemplos: "Praia, Santiago - Cabo Verde", "Mindelo, São Vicente - Cabo Verde"
   - A localização aparecerá nos cartões de imóveis com ícone de mapa vermelho

3. FORMATOS SUPORTADOS:
   - Imagens: JPG, PNG, GIF, WebP
   - Vídeos: MP4, WebM, OGV

4. TAMANHOS RECOMENDADOS:
   - Imagens: 1000x800 pixels ou similar
   - Vídeos: MP4 com boa qualidade

5. FUNCIONALIDADES:
   - Clique nas imagens para visualizar em tela cheia
   - Use os indicadores "Fotos" e "Vídeo" para navegar
   - Localização aparece com ícone vermelho nos cartões
   - Tudo é salvo automaticamente no navegador

6. PARA ADICIONAR MAIS IMÓVEIS:
   - Edite o arquivo src/data/sampleListings.js
   - Ou use o Dashboard como administrador para criar novos imóveis

As imagens, vídeos e localizações aparecerão automaticamente nos cartões de imóveis!
